Simple instructions:
Save these files to a local directory (Git clone)
chmod +x run_virustotal_check.sh

Prerequisites:
1.Install python3
2. Update the "YOUR_API_KEY_HERE" with your virustotal API key
3. Execute the run_virustotal_check.sh
4. Happy hunting
